The problem

Output isn't judgment — and nothing measures the difference.

You commit code, close tickets, ship features. Output has never been higher. But the more polished the AI's output looks, the less you examine it. You stop questioning, stop checking, stop pushing back. People can already feel it.

The trajectory

Atrophy shows up first at the moments that count.

Most builders turned permission prompts off, so the highest-stakes actions — force-push, prod migration, DROP, merge to main — now run with less scrutiny than trivial ones. A dry run over real sessions made it measurable.

The rate you leave the justification blank is the first signal — the wanting-to-own fading, before output quality ever dips. Not a score. A read.

Where we are · 2026

The work has shifted from process to outcome.

With this year's frontier models, the human's contact with the work collapses to three seams: the brief, the mid-flight redirect, the sign-off. Fewer judgment moments, each heavier, none instrumented. The people closest to it are saying it out loud.

The instrument

A Claude Code hook reads each session and writes it down.

Zero commands. No score. It captures what you steered and what you let slide, then surfaces the pattern over time — a mirror for you, not a certificate for someone else.
